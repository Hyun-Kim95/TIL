# 배열

* 복사

```c
int a[SIZE] = {1,2,3,4,5};
int b[SIZE];
int i;

for(i=0;i<SIZE;i++)	// 원소를 일일이 복사한다(통째로는 안됨)
	b[i] = a[i]
```

* 비교
  * 원소를 하나하나 비교해야 함

# 함수

* 배열명은 배열의 주소, 함수명은 함수의 주소

* 인라인 함수
  * 함수 호출 때, 시간이 걸리는데 인라인 함수는 함수를 내부로 가져옴
* 함수 정의를 밑에 하려면 main 함수 위쪽에 함수 원형을 선언해 줘야만 함
