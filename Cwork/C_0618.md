### 소문자를 대문자로 바꾸기

* 아스키 코드를 보면 대문자가 소문자보다 32가 작다
* 즉, 'a' - 32 = 'A'

### 배열의 요소를 계산하는 법

* int size = sizeof(scores) / sizeof(scores[0]);
  * sizeof: 자료형이나 변수의 크기를 바이트 단위로 계산하는 연산자
  * 따라서 요소의 개수 = 전체 바이트 / 배열 요소의 크기
